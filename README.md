# Semestry-app

Make semester workload become a visible heatmap rather than a sudden surpise when all the deadlines come to you and stress you out.

Semestry provides visibility using topic complexity scoring, deadline weight mapping and past papers analysis to help you make better revision, studying and going out guilt free decisions to make university more manageable.


## MVP scope: 
One module. 
Week-by-week heatmap (green/amber/red danger rating). 
Tap a week to get the breakdown — topic complexity, deadline weight, lecture attendance signal, one-line advisory. 
The decision advisor (concept 2) is the main interaction. 
The heatmap is the overview that feeds into it.

Each lecture/week node now has:

✅ Attended / watched recording / skipped
📝 Notes made (yes/no)
🧠 Understanding tag (lost / ok / got it)
🔖 Status tag (done / revisit / make flashcards / needs practice)
